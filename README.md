# 📰 Fake News Detection System

AI-powered fake news classification system built during the Meta PyTorch OpenEnv Hackathon in collaboration with Scaler School of Technology.

This project uses Natural Language Processing (NLP) and machine learning techniques to classify news articles as REAL or FAKE.

---

# 🚀 Features

- Fake vs real news classification
- NLP preprocessing pipeline
- Text tokenization and cleaning
- Machine learning prediction engine
- Interactive Hugging Face deployment
- Fast inference system
- Real-time prediction interface

---

# 🛠️ Tech Stack

## AI/ML
- Python
- Scikit-learn
- NLP preprocessing
- TF-IDF vectorization

## Deployment
- Hugging Face Spaces

## Backend
- Python
- FastAPI

---

# 🧠 Model Pipeline

Input News Article → Text Cleaning → Tokenization → Feature Extraction → ML Classification → Prediction Result
